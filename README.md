# Personal-Library-Manager
Complete library management system with persistent storage
What this project does:
- Add books with ID, Title, Author, Year, and Read Status
- Remove books from library
- Search books by Title or Author
- Show all books in collection
- Show library statistics (total books, read/unread ratio, reading progress)
- Permanently saves data using JSON file handling

**Tech Stack Used:**
Python | File Handling | JSON | CRUD Operations | CustomTkinter (GUI)

**Key Learning Outcomes:**
- Implemented persistent data storage (data doesn't disappear after program closes)
- Built complete CRUD (Create, Read, Update, Delete) operations
- Created both CLI and GUI versions
- Used dictionaries of dictionaries for nested data structure
- Implemented search functionality with multiple criteria
